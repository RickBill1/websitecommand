小鹿窝服务器kanocraft(原MOMB)游戏命令指北

玩家适用

         /back 回死亡点
         /fm 打开设置 GUI
         /fm off 关闭连锁采集
         /fm on 开启连锁采集
         /fm manager 方块管理 GUI（OP）
         /iklog 查看领地审计
         /iklock 打开领地门禁管理面板
         /pland gui 打开领地管理面板
         /pland new default 新建领地
         /pland buy 购买框选的领地范围
         /pland cancel 取消当前选区任务
         /pland set teleport_pos 修改当前领地的传送点为当前所在位置

管理员适用

         /ikban 封禁玩家
         /iklog 查看领地审计
         /iklock 打开领地门禁管理面板
         /lac ban <玩家> [原因] [时长(分钟)] 手动封禁玩家
         /lac unban <玩家> 手动解封玩家
         /lac mute  [duration(minute)] 手动禁言玩家 /lac unmute  解除玩家禁言
         /lac query  查询玩家状态
         /lac reload 重载配置文件
    
         /pland mgr 管理领地
         /pland this 打开当前位置的领地管理GUI
         /ic 查询玩家背包
